Project Description
This project is a static Online Job Portal website created using HTML.
The main purpose of this project is to understand the basics of web development and demonstrate the use of different HTML concepts such as links, tables, lists, images, text formatting, and forms.

The website allows users to:
1)View job listings
2)Register and select job interests
3)Login to the portal
4)Contact the portal for queries

Features
1)Simple navigation using HTML links
2)Job listings displayed using HTML tables
4)User registration and login using HTML forms
5)Job interest selection using checkboxes
6)Images added for better presentation
7)Beginner-friendly and easy to understand code
